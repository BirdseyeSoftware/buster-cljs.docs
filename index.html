buster-cljs

Crossbrowser/Crossplatform Clojurescript testing
Get Started! » Setup Travis CI » Contribute »

Crossplatform

Write a single test suite and run it in both Clojure and Clojurescript.

Crossbrowser

buster.js autoruns your test suite in node.js and slaved browsers (desktop, mobile, phantom.js, etc.).

Clojure.test Compatible

The surface-level API is the same as clojure.test. The JVM version simply wraps clojure.test.
